# Generate Images with Stable Diffusion Forge

This guide explains how to install and configure Stable Diffusion Forge to generate images directly from LibreOffice via Nelson MCP.

## Prerequisites

- Nelson MCP installed in LibreOffice
- Git installed
- A GPU with at least 4 GB VRAM (8+ GB recommended for SDXL models)

## Step 1 — Install Forge

Open **Tools > Options > Nelson MCP > AI Image SD API**.

Click **Detect / Install Forge**.

The installer will:

1. Check available disk space (10-16 GB needed depending on model)
2. Clone Forge from GitHub into `~/stable-diffusion-webui`
3. Create a Python virtual environment
4. Download the selected starter model

### Starter Models

| Model | Quality | VRAM | Download |
|-------|---------|------|----------|
| Juggernaut XL | Best (photorealistic) | 10 GB | 6.5 GB |
| DreamShaper XL | Good (versatile) | 8 GB | 6 GB |
| DreamShaper 8 | Basic (SD 1.5) | 4 GB | 2 GB |

Choose **DreamShaper 8** if you have a modest GPU, **Juggernaut XL** for best quality.

## Step 2 — Launch Forge

Click **Launch Forge** in the options page, or use the menu: **Tools > Nelson MCP > AI Images > Launch Forge**.

Forge opens a terminal window and starts the API server on port 7860. The first launch takes longer as it downloads dependencies.

Once you see `Running on local URL: http://127.0.0.1:7860` in the terminal, Forge is ready.

## Step 3 — Configure an Instance

In **Options > Nelson MCP > AI Image SD API**, you should see an **auto** instance pre-configured by the installer. You can customize it or add more:

- **Endpoint**: `http://127.0.0.1:7860` (default)
- **Model**: leave empty to use the server's current model, or specify a checkpoint filename
- **Resolution**: 512x512 (SD 1.5) or 1024x1024 (SDXL)
- **Steps**: 25 is a good default (6 for Juggernaut XL with LCM)
- **CFG Scale**: 7.0 (lower = more creative, higher = more literal)
- **Sampler**: DPM++ 2M Karras works well for most models

### Hi-Res Fix (optional)

For higher resolution output:

- **Hires Fix**: enable
- **Hires Scale**: 2.0 (doubles resolution)
- **Hires Steps**: 10
- **Hires Denoising**: 0.4

## Step 4 — Generate Images

Use the `generate_image` MCP tool:

> "Generate an image of a sunset over the Mediterranean sea"

The image is generated by Forge and can be automatically inserted into your document or saved to a gallery folder.

### Save to Gallery

In **Options > Nelson MCP > AI Images**, configure **Save to Gallery** to automatically store generated images in a folder gallery.

## Step 5 — CLIP Interrogation

Forge also provides CLIP image interrogation (describing an image in text). This is used by the AI image indexer (Pass 1) to generate initial captions for your photo collection.

No extra setup needed — if Forge is running, CLIP is available.

## Troubleshooting

- **"Connection refused"**: Forge is not running. Click Launch Forge and wait for the terminal to show the URL
- **Out of memory**: Try a smaller resolution or switch to DreamShaper 8 (SD 1.5)
- **Slow generation**: First generation is slower (model loading). Subsequent ones are faster
- **Stop Forge**: Click **Stop Forge** in options or menu to free GPU memory
